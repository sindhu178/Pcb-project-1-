📌 Project Overview

This project is a Dual LED Flasher Circuit PCB designed from scratch as part of my PCB design learning journey.

The goal of this project was to practice:

Schematic creation

Footprint assignment

PCB layout design

Manual routing

Design rule validation


The final design successfully passed:

✅ ERC (Electrical Rule Check) – 0 Errors
✅ DRC (Design Rule Check) – 0 Errors

KiCad (Schematic & PCB Layout)

3D Viewer for visual verification

Through-hole components

Compact board layout

Proper trace clearance and routing

Clearly labeled silkscreen

Verified 3D model preview
